# PIXEL FORGE — ドット絵変換ツール

写真やイラストをレトロなドット絵に変換するPWAアプリです。すべての処理はクライアントサイドのCanvas APIで完結します。

## 機能

- 画像をドラッグ＆ドロップ、またはクリックで選択して読み込み
- **Output Size**: 16×16 〜 256×256px の7段階
- **Palette Preset**: 25種類のプリセットパレット（グループ分け表示）
- **Color Palette**: メディアンカットによる自動色数削減（8〜64色）
- **Illustrate**: アニメ風・フラット・スケッチのイラスト風前処理
- **Outline**: ドット化後の輪郭を強調（Sobel エッジ検出）
- **Dithering**: Floyd-Steinberg 誤差拡散ディザリング
- 変換結果を実ピクセルサイズの PNG としてダウンロード

### パレットプリセット一覧

| グループ | プリセット |
|---|---|
| 基本 | なし（メディアンカット）/ Web Safe / Web Safe 64 / グレースケール 16 / グレースケール 4 |
| レトロゲーム | ゲームボーイ / GB ポケット / GBC パステル / ファミコン (NES) / スーパーファミコン風 / CGA / MSX |
| インディー / モダン | PICO-8 / Sweetie 16 / ENDESGA 32 / DB32 (DawnBringer) / Apollo |
| 芸術 / テーマ | ネオン / サイバーパンク / パステル / アーストーン / オーロラ / セピア / ブルーグレー |

## 変換パイプライン

1. **Illustrate**（任意） — フル解像度でイラスト風処理（彩度調整 → ポスタリゼーション → Sobel エッジ合成）
2. **Downscale** — ニアレストネイバーでターゲットサイズに縮小
3. **Color Reduction** — プリセットパレット適用 or メディアンカット
4. **Outline**（任意） — ドット絵のエッジピクセルをパレット最暗色または純黒で強調
5. **Dithering**（任意） — Floyd-Steinberg 誤差拡散
6. **Display Upscale** — プレビュー表示用のニアレストネイバー拡大（保存には適用されない）

## 開発・実行

ビルド不要。`app/index.html` をブラウザで直接開くだけで動作します。

```
app/
└── index.html   # アプリ本体（単一ファイル）
```

依存ライブラリ・ビルドツール・テストフレームワークはありません。
