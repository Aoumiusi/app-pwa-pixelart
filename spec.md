# PIXEL FORGE — ドット絵変換ツール 仕様書

**バージョン:** 1.0  
**ファイル:** `pixel-art-converter.html`（単一ファイル構成）  
**実装言語:** HTML / CSS / JavaScript（外部ライブラリなし）

---

## 1. 概要

写真・イラスト画像をドット絵（ピクセルアート）に変換するWebツール。  
任意でイラスト化処理を前段に挟むことで、写真→イラスト風→ドット絵の3段階パイプラインを実現する。  
すべての処理はブラウザ内の Canvas API で完結し、サーバー通信は不要。

---

## 2. 変換パイプライン

```
入力画像
  │
  ├─[オプション]─ Step 1: イラスト化
  │                  ポスタリゼーション
  │                  彩度調整
  │                  Sobelエッジ検出 → 輪郭線合成
  │
  ├──────────────── Step 2: ダウンスケール（Nearest Neighbor）
  │                  指定サイズ（16〜256px）に縮小
  │
  ├──────────────── Step 3: 減色処理
  │                  プリセットパレット適用 or メディアンカット
  │
  ├─[オプション]─ Step 4: Floyd-Steinberg ディザリング
  │
  └──────────────── Step 5: 表示用スケールアップ（Nearest Neighbor）
                     プレビュー表示用に等倍〜16x拡大
```

## 3. 機能要件

### 3.1 画像入力

| 項目 | 仕様 |
|---|---|
| 入力方法 | ドラッグ＆ドロップ、クリックしてファイル選択 |
| 対応フォーマット | PNG / JPG / GIF / WebP |
| 最大サイズ制限 | なし（ブラウザのメモリ上限に依存） |

### 3.2 出力サイズ選択（Output Size）

正方形サイズを 7 段階から選択する。デフォルトは 32×32。

| 選択肢 | ピクセル数 |
|---|---|
| 16 × 16 | 256 px |
| **32 × 32**（デフォルト）| 1,024 px |
| 64 × 64 | 4,096 px |
| 96 × 96 | 9,216 px |
| 128 × 128 | 16,384 px |
| 192 × 192 | 36,864 px |
| 256 × 256 | 65,536 px |

### 3.3 パレットプリセット（Palette Preset）

固定色パレットを 7 種類 + なし（自動）から選択する。  
プリセット選択時はメディアンカットを使わず、指定パレットへの最近傍マッピングを行う。

| プリセット名 | 色数 | 説明 |
|---|---|---|
| なし（メディアンカット） | 可変 | 画像から自動生成 |
| ゲームボーイ | 4色 | 実機の緑系4色パレット |
| GB ポケット | 4色 | モノクロ4階調 |
| ファミコン | 54色 | NES PPU の実機パレット |
| CGA / PC-88 風 | 16色 | CGA 標準16色 |
| MSX | 16色 | TMS9918 チップのパレット |
| PICO-8 | 16色 | インディーゲーム向け鮮やか16色 |
| GBC パステル | 32色 | ゲームボーイカラー風パステル |

各プリセットボタンにはカラースウォッチ（代表色）を表示する。

### 3.4 減色数指定（Color Palette）

プリセットが「なし」の場合に有効。メディアンカット法で指定色数に減色する。

| 選択肢 |
|---|
| 無制限（フルカラー） |
| 8色 |
| **16色**（デフォルト） |
| 32色 |
| 64色 |

### 3.5 イラスト化（Illustrate — Step 1）

チェックをオンにすると、ダウンスケール前にフルサイズでイラスト化処理を実行する。

#### スタイル

| スタイル | 処理内容 |
|---|---|
| 🎨 アニメ風 | ポスタリゼーション → Sobel輪郭線 |
| 🟦 フラット | 少ない色段階 + コントラスト強調 → 細い輪郭線 |
| ✏️ スケッチ | 多めの色段階 + 濃い輪郭線 |

#### 調整パラメータ

| パラメータ | 範囲 | デフォルト | 説明 |
|---|---|---|---|
| 輪郭の強さ | 0〜100 | 50 | Sobelエッジの不透明度スケール |
| 色の鮮やかさ | 0〜200 | 130 | 彩度スケール（100=原画、200=2倍） |
| 色の段階数 | 2〜8 | 4 | ポスタリゼーションのレベル数 |

#### イラスト化の処理フロー詳細

```
1. 彩度調整（adjustSaturation）
      各ピクセルを輝度基準にスケール
2. スタイル別処理
      ポスタリゼーション（posterizeData）
      ├─ anime: levels = 設定値
      ├─ flat:  levels = 設定値-1、コントラスト×1.3
      └─ sketch: levels = 設定値+2
3. エッジ検出
      グレースケール変換 → ガウシアンぼかし（3×3 Box blur）
      → Sobelフィルタ（3×3カーネル、Gx・Gy）→ 勾配強度
4. 輪郭線合成
      イラスト化ベース画像にエッジを暗色で乗算合成
```

### 3.6 ディザリング（Dithering — Step 2）

Floyd-Steinberg 誤差拡散法。減色時の色の粗さを目立たなくする。  
イラスト化の後、減色処理と同時に適用される。

誤差拡散の重み配分：

```
       * 7/16
3/16 5/16 1/16
```

### 3.7 表示ズーム（Display Zoom）

プレビュー表示用の拡大倍率。ダウンロードファイルには影響しない。  
大きいサイズ選択時は自動的に上限を制限する。

| 選択肢 | 128px以上 | 192px以上 | 256px |
|---|---|---|---|
| 1x | ○ | ○ | ○ |
| 4x | ○ | ○ | ✗（自動制限） |
| **8x**（デフォルト）| ✗（4x上限）| ✗（2x上限）| ✗ |
| 16x | ✗（4x上限） | ✗（2x上限） | ✗ |

## 6. アルゴリズム詳細

### 6.1 ダウンスケール（Nearest Neighbor）

```javascript
ctx.imageSmoothingEnabled = false;
ctx.drawImage(image, 0, 0, targetSize, targetSize);
```

`imageSmoothingEnabled = false` により補間なしで最近傍ピクセルを選択。  
フォトリアルな縮小ではなくドット感を優先。

### 6.2 メディアンカット法（色数削減）

1. 全ピクセルの RGB 値リストを作成
2. 現在のバケツ群の中から R/G/B の値域が最大のものを選択
3. 最大値域チャンネルで中央値分割
4. バケツ数が目標色数に達するまで 2〜3 を繰り返す
5. 各バケツの平均色をパレット色として採用

### 6.3 最近傍カラーマッピング

各ピクセルとパレット全色のユークリッド距離（RGB空間）を計算し、最小距離の色に置換。

```
distance = (R1-R2)² + (G1-G2)² + (B1-B2)²
```

### 6.4 Sobelエッジ検出

水平・垂直方向の勾配カーネルを適用：

```
Kx = [-1, 0, 1]    Ky = [-1, -2, -1]
     [-2, 0, 2]         [ 0,  0,  0]
     [-1, 0, 1]         [ 1,  2,  1]

G = sqrt(Gx² + Gy²) / 255
```

## 9. 制限事項・既知の挙動

- 出力は正方形固定（非正方形画像は強制的にアスペクト比を変えてリサイズ）
- イラスト化処理は大きい画像（4000px以上）で処理に数秒かかる場合がある
- ガウシアンぼかしは 3×3 Box blur で近似しているため、精密なガウス分布ではない
- ファミコンパレット（54色）は PPU の全パレットエントリを含むが、実機では同時使用は25色まで

```
pixel-art-converter.html   # 全コードを含む単一ファイル
│
├─ <style>                 # CSS（CSS変数、レイアウト、コンポーネント）

│   ├─ <header>            # ロゴ・タイトル

└─ <script>                # JavaScript
    ├─ PRESETS             # パレットプリセット定義
    ├─ buildPresets()      # プリセットUI構築
    ├─ loadFile()          # 画像読み込み
    ├─ btnConvert handler  # 変換メイン処理（async）
    ├─ illustrateImage()   # イラスト化処理
    ├─ adjustSaturation()  # 彩度調整
    ├─ adjustContrast()    # コントラスト調整
    ├─ posterizeData()     # ポスタリゼーション
    ├─ toGrayscale()       # グレースケール変換
    ├─ gaussianBlurData()  # Box blur（3×3）
    ├─ sobelEdge()         # Sobelエッジ検出
    ├─ applyPalette()      # プリセットパレット適用
    ├─ reduceColors()      # メディアンカット減色
    ├─ medianCut()         # メディアンカット法
    ├─ nearestColor()      # 最近傍色マッピング
    └─ floydSteinberg()    # Floyd-Steinbergディザリング
```
